# Special Flags Reference

## When to Escalate to Thurman Immediately

The following situations require immediate flagging — don't wait for the next check-in. Text Thurman directly.

---

## 🚨 Immediate Escalation

### Large Single Event (30+ dozen)
**Why:** Requires special ingredient orders, possible prep help, logistics coordination.

```
⚠️ LARGE EVENT ALERT — [EVENT NAME]
Date: [DATE]
Dozens: [NUM] — this is [X] dozen over our normal range
Event type: [TYPE]
Venue: [VENUE]

Action: Discuss with Thurman before confirming.
```

### First-Time Client
**Why:** No existing relationship to lean on if something goes wrong.

```
⚠️ FIRST-TIME CLIENT — [NAME]
Event: [TYPE] on [DATE]
Source: [HOW FOUND]
Dozens: [NUM]

Action: Flag for extra attention on confirmation and day-of.
```

### Multiple Events Same Weekend
**Why:** Could affect ingredient sourcing, requires schedule coordination.

```
⚠️ CONFLICT — Multiple Events Same Weekend
[DATE 1]: [EVENT NAME] — [DOZENS] dozen — [VENUE]
[DATE 2]: [EVENT NAME] — [DOZENS] dozen — [VENUE]

Action: Check ingredient inventory, confirm both can be fulfilled.
```

### Last-Minute Booking (<48h to Event)
**Why:** Thurman may need to source ingredients fast, confirm availability first.

```
⚠️ LAST-MINUTE INQUIRY — [NAME]
Event date: [DATE] — that's [X] days from now!
Event type: [TYPE]
Dozens: [NUM]

Action: Check with Thurman before sending quote.
```

### Venue Restrictions
**Why:** Could affect setup, delivery, or the actual service.

```
⚠️ VENUE RESTRICTION — [EVENT NAME]
Issue: [NOISE LIMIT / SETUP TIME / PARKING / ACCESS]
Venue: [VENUE]
Event: [DATE]

Action: Discuss with client before confirming.
```

### Undisclosed Dietary Concerns
**Why:** Allergies must be accommodated in prep, could affect flavors.

```
⚠️ DIETARY CONCERN — [EVENT NAME]
Issue: [NUT ALLERGY / GLUTEN / OTHER]
Client: [NAME]
Event: [DATE]

Action: Notify Thurman immediately. Do not confirm until resolved.
```

### Complaint or Negative Feedback
**Why:** Must be handled personally by Thurman.

```
⚠️ COMPLAINT — [PLATFORM]
Client: [NAME]
Event: [DATE]
Rating: [X] stars
Text: "[EXCERPT]"

Action: Thurman handles personally. Do not respond publicly.
```

### Wedding or High-Stakes Corporate
**Why:** High emotional stakes, higher expectation for perfection.

```
⚠️ HIGH-STAKES EVENT — [EVENT NAME]
Type: [WEDDING / CORPORATE / LARGE CELEBRATION]
Date: [DATE]
Dozens: [NUM]
Client: [NAME]

Action: Flag for extra confirmation and day-of attention.
```

### Complaining About Competitors
**Why:** Could indicate a skeptical buyer.

```
⚠️ COMPETITOR MENTION — [NAME]
Mentioned: [COMPETITOR]
Context: "[QUOTE]"

Action: Note in client record. Do not disparage competitor.
```

---

## Flag Format for Thurman

Use this format when texting Thurman:

```
[emoji] [ALERT TYPE] — [EVENT NAME]
[Date] | [Event Type] | [Dozens]
[1-line summary of the issue]
[Urgency level: URGENT / NEED TO DISCUSS / FYI]
```

Keep it short. Thurman will ask questions if he needs them.

---

## Non-Urgent Flags (Send in Daily Digest)

These don't need immediate attention — include in the end-of-day digest:
- Stale inquiry (48h no response)
- Referral lead (flag for follow-up)
- Small order change (add/remove a dozen)
- Weather advisory (mild, no event impact yet)
