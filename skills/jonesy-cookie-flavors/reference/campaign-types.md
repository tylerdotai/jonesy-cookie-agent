# Campaign Types Reference

## Setup for Each Campaign Type

---

## 1. Limited Run Campaign

**Hook:** "Only here through [DATE] — when they're gone, they're gone"

**Purpose:** Create urgency and FOMO. Use for seasonal flavors only. This is the most powerful lever for driving orders.

**Setup:**

1. Set a hard end date when launching
2. Post about it 3 times over the run:
   - Launch day (announce + urgency)
   - Mid-run (reminder + social proof)
   - Last week (final call)
3. Keep it genuinely limited — don't bring it back for 6 months

**Example:**
```
🍪 PUMPKIN SPICE IS BACK — LIMITED THROUGH OCTOBER 31

We're bringing it back. The one you've been waiting for.

When it's gone, it's gone until next fall.

📦 Order by text: (817) 205-1238
📍 Corinth, TX | DFW delivery
```

---

## 2. Flavor Showdown

**Hook:** "First 20 to order get both flavors to compare"

**Purpose:** Drive trial, engagement, conversation. Use when launching two similar flavors or a new flavor against a returning favorite.

**Setup:**

1. Pick two flavors to compare
2. Announce as a "showdown" or "taste test"
3. First N orders get both — creates a cutoff
4. After deadline, announce the winner

**Example:**
```
🍪 FLAVOR SHOWDOWN: Churro vs. Honey Walnut

We're putting two of our most-loved flavors head to head.

First 20 orders this week get a free side-by-side taste test.

Vote for your favorite. Winner stays on the menu.

📦 Order: (817) 205-1238
```

---

## 3. Insider Drop

**Hook:** "24h early access for Jonesy's circle"

**Purpose:** Build community loyalty and an email/SMS list. Reward people who follow on social.

**Setup:**

1. Announce the drop on social with "Insiders only" language
2. Give a 24h window for insider orders
3. Open to general public after 24h
4. Create a simple way to join "the circle" (text list, follow on Instagram)

**Example:**
```
🍪 INSIDER DROP: Holiday Gingerbread — 24h Early Access

Before we announce this one publicly, Jonesy's circle gets first dibs.

Gingerbread is back. In cookie form. Obviously.

Reply "GINGERBREAD" to get on the list.

Open to everyone else Tuesday at 9 AM.
```

---

## 4. Event Tie-In

**Hook:** "Made for [WEDDING SEASON / GAME DAY / GRADUATION] — order by [DATE]"

**Purpose:** Connect flavors to real occasions. Drives advance orders for events.

**Setup:**

1. Identify the event season (wedding, graduation, Super Bowl)
2. Launch a flavor specifically for that season
3. Set an order deadline that ensures delivery in time
4. Announce with event-specific language

**Example:**
```
🍪 MADE FOR GRADUATION SEASON: Party Cookie Platter

Graduation is [DATE]. Your family is coming.

The Party Cookie Platter feeds 20+ — and looks great on the dessert table.

Order by [DEADLINE] for guaranteed delivery before ceremony.

📦 (817) 205-1238
```

## When to Use Each Campaign Type

| Situation | Best Campaign |
|-----------|---------------|
| New seasonal flavor launch | Limited Run + Event Tie-In |
| Two similar flavors competing | Flavor Showdown |
| Building the email/SMS list | Insider Drop |
| Advance orders for a known event | Event Tie-In |
| Slow month, need a boost | Flavor Showdown |
| Holiday gifting push | Limited Run + Gift Box |
| First-time customer conversion | Insider Drop |
