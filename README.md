## What is this?

An OpenClaw AI agent specialized for Jonesy's Cookie Company — handling social media, order inquiries, event logistics, review management, and flavor strategy. Thurman runs a mobile event bakery in Corinth, TX and needs backend leverage to scale without hiring staff.

**Built free as a ClawPlex sponsor benefit.**

---

## Why a public repo?

- Transparent case study for the ClawPlex community
- Real-world proof that OpenClaw agents work for local businesses
- Others can fork and adapt for their own cookie/bakery business
- Thurman gets a professional setup he couldn't build himself

---

## Agent Roles (5 stacks)

| Role | What it does |
|------|-------------|
| **Social Media Manager** | Generates 5-7 posts/week, creates images, writes captions in Jonesy's voice |
| **Order & Inquiry Manager** | Handles incoming messages, qualifies leads, sends quotes, collects deposits |
| **Event & Logistics Coordinator** | Manages event calendar, sends day-of logistics, tracks delivery details |
| **Review & Referral Nurturer** | Monitors reviews, drafts responses, identifies referral opportunities |
| **Flavor Launch Strategist** | Analyzes sales data, proposes seasonal flavors, writes launch copy |
